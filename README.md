# README



## > 開発言語  
* ruby 3.0.1p64  
* Rails 6.0.5

## > 実行手順  
git clone https://github.com/shuheihagino/task2.git  
cd task2_app  
bundle install
yarn install  
rails db:create  
rails db:migrate  
rails db:seed  
rails s
